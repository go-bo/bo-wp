Variables

These global settings are used to keep the look & feel of your web application consistent and easy to maintain.

  • Color Palette
  • Color Defaults
  • Sizing & Spacing
  • Grid Settings
  • Breakpoints
  • Transitions & Timing

Color Palette

The purpose of this HTML is to help determine what default settings are with Bitters and to make sure that all possible HTML Elements are included in this HTML so as to not miss any possible Elements when designing a site.

        
/* 
Color Palette. 
Tints available as a function 
*/
$colors : (
  brand : (
    primary                 : #dfg45a, 
    secondary               : #ffffff,
    accent                  : #ffffff
  ),
  base : (
    light                   : #ffffff,
    dark                    : #000000
  ),
  alert : (
    success                 : #ffffff,
    info                    : #ffffff,
    warning                 : #ffffff,
    error                   : #ffffff
  )
);
        
     


   
/*
Sizing & Spacing.
All layout spacing and ui-sizing should be built on top of this variable map
*/
 $body-background           : color(base,light);
 $text-color                : color(base,dark);
 $user-action-color         : color(brand,primary);
 $user-action-color-hover   : color(brand,primary);
 $border-color              : color(base,light);

/*
Sizing & Spacing.
All layout spacing and ui-sizing should be built on top of this variable map
*/
  $box-sizing               : border-box;
  $base-font-size           : 16px;
  $base-line-height         : 1.5;
  $base-border-radius       : .25rem;
  $base-border-color        : color(base,dark);
  $base-border              : 1px solid $base-border-color;
  $base-spacing             : $base-line-height * 1rem;
  $small-spacing            : $base-spacing / 2;

/* 
Grid Settings.
These are used 
*/
  $grid-columns             : 12;
  $grid-gutter-width        : $base-spacing;
  $enable-flex              : true;
  $enable-grid-classes      : false;
  $max-width                : 1120px;

/* 
Breakpoints.  
If grid classes are enabled, each breakpoint generated about 5kb of css 
*/
$grid-breakpoints: (
  xs                        : 0,      // Extra small screen / phone
  sm                        : 34em,   // Small screen / phone
  md                        : 48em,   // Medium screen / tablet
  lg                        : 62em,   // Large screen / desktop
  xl                        : 75em    // Extra large screen / wide desktop
);

/*
Animation.
Use these settings to keep overall animation feeling consistent.  
*/
  $base-duration            : 250ms !default;
  $base-timing              : ease-out !default;
  $fastDuration             : 100ms !default;
  $mediumDuration           : 300ms !default;
  $slowDuration             : 500ms !default;
  $default-transition-all   : all $base-duration $base-timing;
